Convert RobotFramework output XML to markdown


## Installation

With `pip`:

```bash
pip install robot-markdown
```

With [`pipx`](https://github.com/pipxproject/pipx):

```bash
python3 -m pip install --user pipx
pipx install robot-markdown
```
